# Context for Future Development

## Product Overview
Purple Parrot is a privacy-first baby monitor with:
- **Base unit** - main camera/monitor that sits in nursery
- **Ball** - detachable monitoring sensor that clips magnetically to car seats or prams
- **Link** - handheld device for receiving alerts and viewing video (no phone needed)

## Key Differentiators
- **100% private** - all AI processing on-device (Edge AI), no cloud uploads
- **Portable** - ball detaches for car/pram monitoring with same features
- **No subscription** - one-time purchase, no monthly fees
- **Works without internet** - Link handset connects directly
